# Docker

## Commands
Downloads the docker image.
 ```
 docker pull image
 ```
Runs the docker image.
```
docker run image
```
Listing the running containers.
```
docker ps
```
List all the containers including the exited ones.
```
docker ps -a
```
Stops the container process
```
docker stop IMAGEID
```
```
docker stop NAME
```
Delete a container
```
docker rm IMAGEID
```
